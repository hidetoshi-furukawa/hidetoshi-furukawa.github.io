Apollo

An open autonomous driving platform

Hidetoshi Furukawa
Last updated on May 8, 2020 1 min read
Project

Apollo is an open autonomous driving platform developed by Baidu.

The open software platform is available on Github.

The latest release is Apollo 5.5.

The latest release tag is apollo-v5.5.0. In addition to the tag, there is r5.5.0 branch. The branch continues to make changes for Apollo 5.5.

Apollo 3.5 and later, instead of the Apollo ROS, has been used Apollo Cyber RT.

Caffe is used as a deep learning framework for the perception module. Since Apollo 5.0, Baidu’s PaddlePaddle (PArallel Distributed Deep LEarning) is also supported.
